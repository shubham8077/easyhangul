Basic Rules

Hangul, the Korean writing system, follows specific rules for placing consonants and vowels to form syllables. Here are the basic points of these rules:

Syllable Block Structure | ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§¨‡•ç‡§≤‡•â‡§ï ‡§∏‡§Ç‡§∞‡§ö‡§®‡§æ:

- Each Hangul syllable is written as a block that combines at least one consonant and one vowel.
- A syllable block can consist of 2 to 4 letters.

- ‡§™‡•ç‡§∞‡§§‡•ç‡§Ø‡•á‡§ï ‡§π‡§æ‡§Ç‡§ó‡•Å‡§≤ ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§ï‡•ã ‡§è‡§ï ‡§¨‡•ç‡§≤‡•â‡§ï ‡§ï‡•á ‡§∞‡•Ç‡§™ ‡§Æ‡•á‡§Ç ‡§≤‡§ø‡§ñ‡§æ ‡§ú‡§æ‡§§‡§æ ‡§π‡•à ‡§ú‡§ø‡§∏‡§Æ‡•á‡§Ç ‡§ï‡§Æ ‡§∏‡•á ‡§ï‡§Æ ‡§è‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§î‡§∞ ‡§è‡§ï ‡§∏‡•ç‡§µ‡§∞ ‡§π‡•ã‡§§‡§æ ‡§π‡•à‡•§
- ‡§è‡§ï ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§¨‡•ç‡§≤‡•â‡§ï ‡§Æ‡•á‡§Ç 2 ‡§∏‡•á 4 ‡§Ö‡§ï‡•ç‡§∑‡§∞ ‡§π‡•ã ‡§∏‡§ï‡§§‡•á ‡§π‡•à‡§Ç‡•§


Initial Consonant (Choseong) | ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§®:

- Every syllable block must start with an initial consonant, which can be any of the 14 basic consonants („Ñ±,„Ñ¥,„Ñ∑, etc.).
- If a syllable starts with a vowel sound, the silent consonant "„Öá" is used as a placeholder.

- ‡§™‡•ç‡§∞‡§§‡•ç‡§Ø‡•á‡§ï ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§¨‡•ç‡§≤‡•â‡§ï ‡§ï‡•ã ‡§è‡§ï ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§∏‡•á ‡§∂‡•Å‡§∞‡•Ç ‡§π‡•ã‡§®‡§æ ‡§ö‡§æ‡§π‡§ø‡§è, ‡§ú‡•ã 14 ‡§Æ‡•å‡§≤‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§®‡•ã‡§Ç („Ñ±, „Ñ¥, „Ñ∑, ‡§Ü‡§¶‡§ø) ‡§Æ‡•á‡§Ç ‡§∏‡•á ‡§ï‡•ã‡§à ‡§≠‡•Ä ‡§π‡•ã ‡§∏‡§ï‡§§‡§æ ‡§π‡•à‡•§
- ‡§Ø‡§¶‡§ø ‡§è‡§ï ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§∏‡•ç‡§µ‡§∞ ‡§ß‡•ç‡§µ‡§®‡§ø ‡§∏‡•á ‡§∂‡•Å‡§∞‡•Ç ‡§π‡•ã‡§§‡•Ä ‡§π‡•à, ‡§§‡•ã ‡§Æ‡•Ç‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® "„Öá" ‡§ï‡§æ ‡§â‡§™‡§Ø‡•ã‡§ó placeholder ‡§ï‡•á ‡§∞‡•Ç‡§™ ‡§Æ‡•á‡§Ç ‡§ï‡§ø‡§Ø‡§æ ‡§ú‡§æ‡§§‡§æ ‡§π‡•à‡•§


Medial Vowel (Jungseong) | ‡§Æ‡§ß‡•ç‡§Ø‡§µ‡§∞‡•ç‡§§‡•Ä ‡§∏‡•ç‡§µ‡§∞:

- The medial vowel follows the initial consonant.
* Vowels are divided into two types based on their shape:
- Vertical vowels („Öè, „Öë, „Öì, „Öï, „Ö£) are placed to the right of the initial consonant.
- Horizontal vowels („Öó, „Öõ, „Öú, „Ö†, „Ö°) are placed below the initial consonant.

- ‡§Æ‡§ß‡•ç‡§Ø‡§µ‡§∞‡•ç‡§§‡•Ä ‡§∏‡•ç‡§µ‡§∞ ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§ï‡•á ‡§¨‡§æ‡§¶ ‡§Ü‡§§‡§æ ‡§π‡•à‡•§
* ‡§∏‡•ç‡§µ‡§∞ ‡§â‡§®‡§ï‡•á ‡§Ü‡§ï‡§æ‡§∞ ‡§ï‡•á ‡§Ü‡§ß‡§æ‡§∞ ‡§™‡§∞ ‡§¶‡•ã ‡§™‡•ç‡§∞‡§ï‡§æ‡§∞‡•ã‡§Ç ‡§Æ‡•á‡§Ç ‡§µ‡§ø‡§≠‡§æ‡§ú‡§ø‡§§ ‡§π‡•ã‡§§‡•á ‡§π‡•à‡§Ç:
- ‡§ä‡§∞‡•ç‡§ß‡•ç‡§µ‡§æ‡§ß‡§∞ ‡§∏‡•ç‡§µ‡§∞ („Öè, „Öë, „Öì, „Öï, „Ö£) ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§ï‡•á ‡§¶‡§æ‡§à‡§Ç ‡§ì‡§∞ ‡§∞‡§ñ‡•á ‡§ú‡§æ‡§§‡•á ‡§π‡•à‡§Ç‡•§
- ‡§ï‡•ç‡§∑‡•à‡§§‡§ø‡§ú ‡§∏‡•ç‡§µ‡§∞ („Öó, „Öõ, „Öú, „Ö†, „Ö°) ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§ï‡•á ‡§®‡•Ä‡§ö‡•á ‡§∞‡§ñ‡•á ‡§ú‡§æ‡§§‡•á ‡§π‡•à‡§Ç‡•§


Final Consonant (Jongseong) | ‡§Ö‡§Ç‡§§‡§ø‡§Æ ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§®:

- A syllable may end with a final consonant, which is placed at the bottom of the syllable block.
- Final consonants can be any of the 14 basic consonants or combinations known as double consonants („Ñ≤, „Ñ∏, etc.).

- ‡§è‡§ï ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§è‡§ï ‡§Ö‡§Ç‡§§‡§ø‡§Æ ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§ï‡•á ‡§∏‡§æ‡§• ‡§∏‡§Æ‡§æ‡§™‡•ç‡§§ ‡§π‡•ã ‡§∏‡§ï‡§§‡•Ä ‡§π‡•à, ‡§ú‡•ã ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§¨‡•ç‡§≤‡•â‡§ï ‡§ï‡•á ‡§®‡•Ä‡§ö‡•á ‡§∞‡§ñ‡§æ ‡§ú‡§æ‡§§‡§æ ‡§π‡•à‡•§
- ‡§Ö‡§Ç‡§§‡§ø‡§Æ ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® 14 ‡§Æ‡•å‡§≤‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§®‡•ã‡§Ç ‡§Æ‡•á‡§Ç ‡§∏‡•á ‡§ï‡•ã‡§à ‡§≠‡•Ä ‡§π‡•ã ‡§∏‡§ï‡§§‡•á ‡§π‡•à‡§Ç ‡§Ø‡§æ ‡§°‡§¨‡§≤ ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§ï‡•á ‡§∞‡•Ç‡§™ ‡§Æ‡•á‡§Ç ‡§ú‡§æ‡§®‡•á ‡§ú‡§æ‡§®‡•á ‡§µ‡§æ‡§≤‡•á ‡§∏‡§Ç‡§Ø‡•ã‡§ú‡§® („Ñ≤, „Ñ∏, ‡§Ü‡§¶‡§ø) ‡§π‡•ã ‡§∏‡§ï‡§§‡•á ‡§π‡•à‡§Ç‡•§


Basic Syllable Block Formations | ‡§Æ‡•Ç‡§≤ ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§¨‡•ç‡§≤‡•â‡§ï ‡§∏‡§Ç‡§∞‡§ö‡§®‡§æ‡§è‡§Ç:

- Consonant + Vowel (CV): If the vowel is vertical, the block is written side by side (e.g., Í∞Ä, ÎÇò, Îã§).

- ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® + ‡§∏‡•ç‡§µ‡§∞ (CV): ‡§Ø‡§¶‡§ø ‡§∏‡•ç‡§µ‡§∞ ‡§ä‡§∞‡•ç‡§ß‡•ç‡§µ‡§æ‡§ß‡§∞ ‡§π‡•à, ‡§§‡•ã ‡§¨‡•ç‡§≤‡•â‡§ï ‡§ï‡•ã ‡§∏‡§æ‡§•-‡§∏‡§æ‡§• ‡§≤‡§ø‡§ñ‡§æ ‡§ú‡§æ‡§§‡§æ ‡§π‡•à (‡§ú‡•à‡§∏‡•á, Í∞Ä, ÎÇò, Îã§)‡•§

- If the vowel is horizontal, the block is written top to bottom (e.g., Í≥†, ÎÖ∏, ÎèÑ).

- ‡§Ø‡§¶‡§ø ‡§∏‡•ç‡§µ‡§∞ ‡§ï‡•ç‡§∑‡•à‡§§‡§ø‡§ú ‡§π‡•à, ‡§§‡•ã ‡§¨‡•ç‡§≤‡•â‡§ï ‡§ï‡•ã ‡§ä‡§™‡§∞ ‡§∏‡•á ‡§®‡•Ä‡§ö‡•á ‡§≤‡§ø‡§ñ‡§æ ‡§ú‡§æ‡§§‡§æ ‡§π‡•à (‡§ú‡•à‡§∏‡•á, Í≥†, ÎÖ∏, ÎèÑ)‡•§

- Consonant + Vowel + Consonant (CVC): The final consonant is added below the initial consonant and vowel (e.g., Í∞ï, ÎÑê, Îã≠).

- ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® + ‡§∏‡•ç‡§µ‡§∞ + ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® (CVC): ‡§Ö‡§Ç‡§§‡§ø‡§Æ ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§ï‡•ã ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§î‡§∞ ‡§∏‡•ç‡§µ‡§∞ ‡§ï‡•á ‡§®‡•Ä‡§ö‡•á ‡§ú‡•ã‡§°‡§º‡§æ ‡§ú‡§æ‡§§‡§æ ‡§π‡•à (‡§ú‡•à‡§∏‡•á, Í∞ï, ÎÑê, Îã≠)‡•§

- Consonant + Vowel + Consonant + Consonant (CVCC): Some syllables can end with double consonants, placed together below the initial consonant and vowel (e.g., Ïïâ, ÏûÉ).

- ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® + ‡§∏‡•ç‡§µ‡§∞ + ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® + ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® (CVCC): ‡§ï‡•Å‡§õ ‡§µ‡§∞‡•ç‡§£‡§Æ‡§æ‡§≤‡§æ ‡§°‡§¨‡§≤ ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§®‡•ã‡§Ç ‡§ï‡•á ‡§∏‡§æ‡§• ‡§∏‡§Æ‡§æ‡§™‡•ç‡§§ ‡§π‡•ã ‡§∏‡§ï‡§§‡•Ä ‡§π‡•à‡§Ç, ‡§ú‡•ã ‡§™‡•ç‡§∞‡§æ‡§∞‡§Ç‡§≠‡§ø‡§ï ‡§µ‡•ç‡§Ø‡§Ç‡§ú‡§® ‡§î‡§∞ ‡§∏‡•ç‡§µ‡§∞ ‡§ï‡•á ‡§®‡•Ä‡§ö‡•á ‡§è‡§ï ‡§∏‡§æ‡§• ‡§∞‡§ñ‡•á ‡§ú‡§æ‡§§‡•á ‡§π‡•à‡§Ç (‡§ú‡•à‡§∏‡•á, Ïïâ, ÏûÉ)‡•§


Examples | ‡§â‡§¶‡§æ‡§π‡§∞‡§£:

- Í∞Ä (ga): „Ñ± (initial) + „Öè (medial)
- Í∞ï (gang): „Ñ± (initial) + „Öè (medial) + „Öá (final)
- ÌïôÍµê (hak-gyo): Ìïô („Öé + „Öè + „Ñ±) + Íµê („Ñ± + „Öõ)
- Ïó¨ (yeo): „Öá (initial, silent) + „Öï (medial)
- Ïùë (eung): „Öá (initial) + „Ö° (medial) + „Öá (final)


By following these rules, you can form various syllables and words in Hangul.
